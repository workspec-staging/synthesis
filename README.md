# fullstack-typescript-project

A minimal Aspire TypeScript project for WorkSpec validation. Runs a Vite web app via `aspire run`.

## Prerequisites

- Node.js 22+
- .NET 10 SDK
- Aspire CLI (`dotnet tool install -g aspire`)
- pnpm (`npm install -g pnpm`)

## Run locally

```bash
aspire run
```

Aspire restores the SDK bindings, installs web app dependencies via pnpm, and starts the Vite dev server. It prints the dashboard URL and the web app URL on startup.

## What it does

- `apphost.ts` — declares a single Vite web app resource
- `artifacts/web/` — minimal Vite + TypeScript app showing "Hello WorkSpec", the bound port, and build timestamp
- `aspire.config.json` — pins the Aspire SDK version and packages; `aspire restore` regenerates `.modules/` from this

## Project structure

```
fullstack-typescript-project/
├── apphost.ts             # Aspire AppHost — declares resources
├── aspire.config.json     # Aspire SDK version + packages (drives aspire restore)
├── artifacts/
│   └── web/               # Vite + TypeScript web app
└── pnpm-workspace.yaml
```

`.modules/` is gitignored — it is generated by `aspire restore` and must not be committed.
